欢迎页 >>系统说明

服务端：

Spring Boot2.1.3.RELEASE+Mybatis3.3.0+druid1.1.9等

前端：

JQuery+Bootstrap3.3.5+Start Bootstrap-SB Admin 2（基于bootstrap的响应式后台管理模板）等

数据存储：

数据库使用Mysql

主要功能：

系统功能：会员建档管理、课表铺排、专业教练服务、健身计划及健身记录、会员特权服务、定期问卷调查及服务升级等

日志信息管理：用户登录信息、用户操作信息

多级菜单：最多支持二级菜单

控制台：欢迎页

特点：

1、简单，项目代码均添加注释，阅读方便

2、精简，采用经典的MVC模式，对数据访问层和业务逻辑层进行了抽象，大大提高开发效率

3、项目使用maven进行构建

4、使用CDN加速，更快的页面加载体验

5、使用bootstrap响应式布局，PC、移动端完美自适应

6、地址栏智能过滤，防止用户直接输入地址进行操作

7、404,500页面

8、基于mybatis-generator代码自动生成

9、Echarts 4.2.1 图表应用

计划中功能:

10、shiro

11、基于apidoc接口文档自动生成

部署方法：cmd java -jar jianshen.jar

mybatisgenerator方法：mvn mybatis-generator:generate
